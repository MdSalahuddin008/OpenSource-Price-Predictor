# Open Source Price Predictor 💰🤖

Fine-tunes an open-source LLM (**Llama 3.2 3B**) with QLoRA to predict a product's price from its text description — then benchmarks it against a frontier model (GPT-4.1-nano) and human-level performance.

👉 Full project, setup instructions, and file breakdown are in [`open-source-price-predictor/`](./open-source-price-predictor).
